# Deity Images Setup

Please add the following deity images to this directory (`public/lovable-uploads/`):

## Required Images:

1. **shiva.jpg** - Shiva deity image (from chat attachment #1)
2. **ganesh.jpg** - Ganesh deity image (from chat attachment #2)
3. **sri-krishna.jpg** - Sri Krishna deity image (from chat attachment #3)
4. **bhairava.jpg** - Bhairava deity image (from chat attachment #4)
5. **devi.jpg** - Devi deity image (from chat attachment #5)

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 800x1200px or similar aspect ratio
- Should have good lighting for text overlay readability

## How to Add:
1. Save each deity image from the chat attachments
2. Rename them according to the list above
3. Place them in this directory: `public/lovable-uploads/`
4. The WorkspaceSection will automatically load them

## Fallback:
If images fail to load, the component will fall back to gradient backgrounds with the appropriate deity colors.

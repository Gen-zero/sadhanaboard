SUPPORT & HELP
================================================================================

If You Get Stuck:

1. Check the Troubleshooting Guide:
   MONGODB_QUICKSTART.txt - Section 12

2. Run the Test Script:
   node backend/test-mongodb-connection.js

3. Check the Server:
   npm run dev (watch console for errors)

4. Review Migration Plan:
   MONGODB_MIGRATION_PLAN.txt (find relevant section)

5. Check Health Endpoints:
   curl http://localhost:3004/api/health/db

6. Check Logs:
   Look at console output from npm run dev

Common Issues:

Issue: "Cannot find module 'mongoose'"
  Solution: npm install mongoose

Issue: Connection timeout
  Solution: Check MongoDB Atlas IP whitelist

Issue: Server won't start
  Solution: Check .env for MONGODB_URI

Issue: Need quick command
  Solution: MONGODB_QUICKSTART.txt - Section 8
